## Manual Restore

Files are stored in their original directory structure. Copy what you need:

```bash
# Example: restore just handshakes
sudo cp -r home/pi/handshakes /home/pi/

# Example: restore config
sudo cp -r etc/pwnagotchi /etc/
```
